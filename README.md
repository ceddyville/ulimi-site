# Ulimi Site

Frontend for [ulimi.dev](https://ulimi.dev) — The African Tongue.

## Files

- `index.html` — Client-facing dictionary (earth tones, Cormorant Garamond)
- `docs.html` — Developer API documentation (dark green, DM Mono)

## Status

HTML templates — to be ported to Next.js.

## API

Connects to [ulimi-api](https://github.com/ceddyville/ulimi-api) at `ulimi.dev/api/v1/`.
